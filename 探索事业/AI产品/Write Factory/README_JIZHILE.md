# 极致了API配置说明

## 概述

内容工厂现已集成"极致了"数据平台API，用于微信公众号文章数据采集和选题分析。

## 数据源信息

**平台名称**: 极致了
**官方网站**: https://www.jizhile.com
**功能描述**: 微信公众号文章数据采集与分析平台
**数据类型**: 公众号文章、阅读量、点赞数、互动率等

## 配置步骤

### 1. 获取API Key

1. 访问 [极致了官网](https://www.jizhile.com)
2. 注册并登录账号
3. 在个人中心或API管理页面找到API Key
4. 复制您的专属API Key

### 2. 在系统中配置

1. 登录内容工厂
2. 进入 "设置" 页面
3. 在"数据源配置"中选择"极致了"
4. 输入您的API Key并保存

### 3. 开始使用

- 前往"选题分析"页面
- 输入关键词进行分析
- 系统将自动调用极致了API获取数据

## API调用参数

系统会自动配置以下参数：

```javascript
{
  "kw": "搜索关键词",
  "sort_type": 1,        // 排序：1-最新，2-最热
  "mode": 1,            // 匹配：1-精确，2-模糊
  "period": 7,          // 时间周期（天）
  "page": 1,             // 页码
  "key": "用户API Key",   // 您的API Key
  "type": 1,            // 内容类型：1-文章
  "limit": 20           // 每页数量
}
```

## 数据字段说明

返回数据包含以下主要字段：

- `title`: 文章标题
- `read`: 阅读量
- `praise`: 点赞数
- `wx_name`: 公众号名称
- `publish_time_str`: 发布时间
- `url`: 文章链接
- `short_link`: 短链接

## 使用限制

- 需要有效的极致了API Key
- 遵守极致了平台使用协议
- API调用频率限制以极致了平台为准

## 故障排除

### API Key错误
- 确认API Key正确无误
- 检查API Key是否有效
- 联系极致了客服

### 数据获取失败
- 检查网络连接
- 确认关键词格式正确
- 尝试使用其他关键词

### 账号问题
- 确保极致了账号正常
- 检查API调用额度
- 联系平台管理员

## 技术支持

如遇技术问题，请检查浏览器控制台错误信息或联系开发团队。
