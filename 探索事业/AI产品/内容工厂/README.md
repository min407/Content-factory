## 📝 项目说明
这是一个基于Next.js的AI内容创作平台，支持选题分析、内容创作和多平台发布。

### 主要功能
- 🔍 **选题分析**: AI智能分析公众号文章，生成选题洞察
- ✍️ **内容创作**: 基于选题自动生成高质量内容
- 📱 **多平台发布**: 支持微信公众号、小红书等平台发布
- 📊 **数据统计**: 实时追踪内容表现和互动数据

### 技术栈
- **前端**: Next.js 16, React 19, TypeScript
- **样式**: Tailwind CSS, Radix UI
- **数据库**: SQLite, better-sqlite3
- **图表**: Recharts
- **图标**: Lucide React
